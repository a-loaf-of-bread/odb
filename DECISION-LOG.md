#### Decision: (20th May 2026)
> Create a file COMPLIANCE.md and update CONTRIBUTING.md to ensure that the repository is compliant with the GitHub Acceptable Use Policies and all contributors are aware of the necessities of this repository and the platform hosting it

Reasoning:
- Islamically speaking, when you enter a contract (or in this case, click "Agree" on a certain platform, honoring those terms becomes a binding obligation. (refer to Surah al Ma'ida, Ayah 1) 
- The repository is hosted on GitHub and therefore must comply with the rules set forth by GitHub (ie the rules that were agreed to upon signing up)
- The acceptable use policies set high standards, which is precisely what this repo already aimed for (such as intellectual honesty and rigorous proof)
- The core objective of the acceptabe use policies happens to match Islamic principles and teachings in nature

#### Decision: (9th May 2026)
> To create a "decision log" file in the rough-bread branch

Reasoning:
- To ensure absolute transparency and to ensure that no one has to assume so as to 'why' a decision was made
- Whilst the GitHub commit messages are good, a proper Architecture Design Record file is far more accessible and easier for people to read
- To make the project easier to continue by potentially increasing the bus factor by documenting the decision making process of the Loaf of Bread

#### Decision: (9th May 2026)
> To change the license from CC-BY-NC SA 4.0 to CC-BY-SA 4.0

Reasoning:
- Sometimes even YouTube videos can be considered as "commercial" 
- If the project becomes successful (InshaAllah) then people/orgs will need to be contacted for covering costs of printing, designing, translating, etc
- It still remains free as any modified version must be made available under the same license

#### Decision: (19th March 2026)
> To license the project under CC BY NC SA 4.0

Reasoning:
- Licensed the project under CC-BY-NC-SA 4.0 in order to permit viewers to utilise the content of this project for non-commercial purposes.
- to ensure that the project can be continued even if the Loaf of Bread is no longer present, 
- to ensure that the project can never be monetised since it has to be licensed under the same license even if modified,
- to ensure that the project isn't used for the financial profit of anyone and stays a project for the ummah,
- to ensure that this project is mentioned if it's content is used to get more contributors
